Lectures

If you are a teacher using this course for a classroom or online setting, you may find these lecture slides helpful.

Each lecture should take about 1 hour to present. The unit lectures cover at least the first topics in a unit, but don't always cover all the topics in the unit, as students are expected to work through the rest of the unit on their own. The bonus lectures cover additional topics that are not covered in the articles.

The slides are written in HTML/CSS using Reveal.JS, so that they can be viewed in a web browser. Download the original ones from the GitHub repository if you'd like to edit them.

  • Unit 1: Expressions & functions
  • Bonus topic: Debugging
  • Unit 2: Loops & Lists
  • Bonus topic: Testing
  • Unit 3: Strings & Dictionaries
  • Unit 4: Object Oriented Programming
  • Bonus topic: Workflow
